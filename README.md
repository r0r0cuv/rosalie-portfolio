# 🌐 Rosalie Reblora Portfolio Website

## 📌 Overview
This is my personal portfolio website built using HTML, CSS, and JavaScript. It showcases my background in Computer Science, IT systems support, cybersecurity, and software development projects.

The goal of this website is to present my skills, experience, and projects in a clean and professional format for recruiters and employers.

---

## 🚀 Features
- Responsive and modern design  
- Smooth scrolling and animations  
- Organized sections for:
  - About Me  
  - Education  
  - Skills  
  - Experience  
  - Projects  
  - Contact  
- Interactive UI with JavaScript  
- Deployed using GitHub Pages
